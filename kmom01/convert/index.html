Instuderingsfrågor om JavaScript

  • Ge en kort historik över programmeringsspråket JavaScript.

    • JavaScript skapades år 1995 av Brendan Eich på företaget Netscape.

    • Ursprungligen hette det Mocha, sedan LiveScript, innan det fick namnet JavaScript.
    • Det utvecklades för att göra webbsidor mer interaktiva, något som HTML och CSS inte kunde göra själva.
    • rots namnet har språket ingen koppling till Java, namnet valdes mest för marknadsföring.
    • Numera är JavaScript ett av de mest använda språken i världen och körs i alla webbläsare, men också på servrar via Node.js.
